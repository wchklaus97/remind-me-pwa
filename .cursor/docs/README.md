# Documentation Archive

This directory contains architecture, migration, and planning documentation for the Remind Me PWA project.

## Directory Structure

### `architecture/`
Architecture decision records and overview documents:
- `ARCHITECTURE_DECISION.md` - Architecture decisions and rationale
- `ARCHITECTURE_OVERVIEW.md` - Overall system architecture
- `BLOC_ARCHITECTURE_PLAN.md` - BLOC pattern architecture plans (historical)
- `UI_COMPONENT_ARCHITECTURE.md` - UI component architecture patterns

### `migrations/`
Migration guides and upgrade documentation:
- `DIOXUS_0.7_UPGRADE.md` - Dioxus 0.7 upgrade guide
- `FULL_STACK_MIGRATION_GUIDE.md` - Full-stack migration plans (historical)
- `FIXING_COMPONENTS.md` - Component fixes and refactoring notes
- `I18NRS_MIGRATION.md` - i18nrs migration guide

### `plans/`
Development plans and project documentation:
- `DEVELOPMENT_PLAN.md` - Development roadmap
- `IMPLEMENTATION_PLAN.md` - Implementation details
- `MODULE_STRUCTURE.md` - Module organization patterns
- `PR_DESCRIPTION.md` - Pull request descriptions
- `PROJECT_REVIEW.md` - Project review and status
- `REFACTORING_COMPLETE.md` - Refactoring completion notes
- `REFACTORING_PLAN.md` - Refactoring plans
- `UI_COMPONENTS_IMPLEMENTATION.md` - UI components implementation guide
